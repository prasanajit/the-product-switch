## Q1. What is an Algorithm?

Algorithm is basically the logic you put up to solve a problem.

## Q2. Why do we calculate Time Complexity?

We calculate time complexity to determine which solution of a problem is better, there could be multiple solutions, so it helps identifying those.

## Q3. Why is O(n) better than O(n²)?

O(n) takes n number of operations on a single run but O(n²) does n² operations on a single run so O(n) is better compared to O(n²).

## Q4. Is O(100n) better than O(n²)? Why?

Yes, O(100n) is better than O(n²) because it still O(n) and it does depends on n and it does n number of calculations while in O(n²) its n² calculation which takes more time and adds up the complexity.
