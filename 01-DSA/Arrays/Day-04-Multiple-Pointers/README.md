1. What is the Multiple Pointers Pattern?

Multiple pointer pattern where we take two variables where we can do operations with it and give a required output

2. Why do we use the Multiple Pointers Pattern?

We use the Multiple Pointers pattern to reduce time complexity by avoiding unnecessary comparisons, especially on sorted data.

3. When should we use this pattern?

to avoid multiple loops 

4. What type of array does this pattern usually work best on?

sorted array

5. Why is a sorted array important for many Multiple Pointer problems?

Many Multiple Pointer problems rely on sorted arrays because the order allows pointers to move intelligently. However, not every Multiple Pointer problem requires a sorted array.

8. Advantages of the Multiple Pointers Pattern.

We avoid repeated comparisons and reduce unnecessary iterations.

9. Disadvantages of the Multiple Pointers Pattern.

Many interview problems use sorted arrays, but the pattern can also be applied to certain unsorted problems depending on the logic.
